Subschema Simple Example
====

Shows how to use subschema in a most minimal way.

You are better off using one of the generated examples from the subschema [demo](http://subschema.github.io/subschema)

You can see the very basic demo [here](http://subschema.github.io/subschema-simple-example)

##Installation
To install and run

```sh
 $ git clone https://github.com/jspears/subschema-simple-example
 $ cd subschema-simple-example
 $ npm i
 $ npm run hot
 $ open -a Chrome http://127.0.0.1:8007/

```
